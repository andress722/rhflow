# UI Guidelines — PresençaFlow RH

## Tom visual

B2B, limpo, operacional e objetivo. Evitar aparência de app genérico de RH colorido demais.

## Status colors sugeridos

- OPEN: azul/cinza
- WAITING_HR: amarelo
- WAITING_MANAGER: roxo
- WAITING_EMPLOYEE: laranja
- RESOLVED: verde
- REJECTED: vermelho
- CANCELLED: cinza

## Componentes shadcn/ui recomendados

- Card
- Table
- Badge
- Dialog
- Drawer
- Sheet
- Form
- Input
- Select
- Tabs
- Calendar
- DropdownMenu
- Toast/Sonner

## Priorização de UX

- RH precisa ver pendências primeiro.
- Gestor precisa ver equipe dele primeiro.
- Funcionário não usa painel no MVP, usa WhatsApp.
- Atestado precisa abrir rápido e com preview.
- Timeline deve estar sempre visível no detalhe da ocorrência.
