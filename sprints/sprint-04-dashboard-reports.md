# Sprint 04 — Dashboard e Relatórios

## Objetivo

Criar visão gerencial para RH e exportações simples para fechamento de DP/folha.

## Entregas

- Dashboard RH.
- Dashboard gestor com escopo da equipe.
- KPIs principais.
- Relatórios filtráveis.
- Exportação CSV/Excel.
- Logs de exportação.

## Backend tasks

- Criar endpoints `/api/dashboard/summary`, `/today`, `/pending`.
- Criar relatórios de ocorrências, ausências e atestados.
- Criar exportação CSV.
- Opcional: Excel com ExcelJS.
- Registrar ExportLog.

## Frontend tasks

- Cards de KPI.
- Tabela de pendências.
- Relatórios com filtros.
- Botão exportar.
- Tela de pendências por gestor.

## Critérios de aceite

- RH vê indicadores do dia.
- Gestor vê apenas sua equipe.
- Relatório exporta ocorrências por período.
- Exportação gera log.
